Pantalla para engadir exemplar

Esta pestana permíte engadir ou modificar unha instancia dunha banda deseñada.
Para que o exemplar sexa válido, debe ter as seguintes características:

  • Unha data de adquisición que xa ocorreu
  • Un prezo numérico superior a 0,00 e inferior ou igual a 9.999.999
  • Un límite e un estado que teñen entre 1 e 50 caracteres cada un
