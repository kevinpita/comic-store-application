Pantalla para añadir ejemplar

Esta pestaña permite agregar o modificar un ejemplar de un cómic.
Para que el ejemplar sea válido tiene que tener las siguientes características:

  • Una fecha de adquisición que ya haya sucedido
  • Un precio numérico mayor que 0.00 y menor o igual que 9 999 999
  • Una tapa y un estado que tengan entre 1 y 50 caracteres cada uno
