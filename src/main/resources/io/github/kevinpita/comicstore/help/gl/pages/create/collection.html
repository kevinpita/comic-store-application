Pantalla para crear unha colección

Esta pestana permíte engadir, eliminar ou modificar unha colección.
Para que a colección sexa válida debe ter as seguintes características:

  • Un nome único
  • O nome e o editor teñen entre 1 e 255 caracteres cada un
  • A descrición non supera os 512 caracteres

Se se usa como edición, estará dispoñible a opción de eliminación
e tamén mostrará as bandas deseñadas asociados a esa colección.
