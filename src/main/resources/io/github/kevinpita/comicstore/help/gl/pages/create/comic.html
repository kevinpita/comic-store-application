Pantalla para crear unha banda deseñada

Esta pestana permíte engadir, eliminar ou modificar unha banda deseñada.
Para que a banda deseñada teña validez, debe ter as seguintes características:

  • O nome ten entre 1 e 255 caracteres
  • O número de publicación está entre 1 e 999 999 999
  • Debe ter unha colección
  • O número de publicación ten que ser único para a colección
  • A descrición non supera os 512 caracteres

Se se usa como edición, a opción de eliminar estará dispoñible
e tamén mostrará os exemplares e autores asociados

Para crear, editar e eliminar os exemplares e os autores
asociados a unha banda deseñada, hai dúas táboas unha táboa para os exemplares
e outra para os autores.

Xunto a cada unha das dúas táboas hai 3 botóns,
Novo, Editar e Eliminar, que permiten crear, modificar e eliminar exemplares ou autores asociados ao cómic.
