Pantalla de ajustes del servidor

Esta pantalla permite cambiar la dirección del servidor y la contraseña que se usa para acceder.
La dirección del servidor debe de incluir el protocolo http:// o https:// y el nombre o ip del servidor.
Los datos se pueden comprobar mediante el botón de comprobar o mediante el botón de guardar,
el cual los comprobará antes de guardarlos.
