Pantalla para crear una colección

Esta pestaña permite agregar, borrar o modificar una colección.
Para que la colección sea válida tiene que tener las siguientes características:

  • Un nombre único
  • El nombre y el publicador tienen entre 1 y 255 caracteres cada uno
  • La descripción no tiene más de 512 caracteres

Si se usa como edición, la opción de borrar estará disponible
y también visualizará los cómics asociados a dicha colección.
