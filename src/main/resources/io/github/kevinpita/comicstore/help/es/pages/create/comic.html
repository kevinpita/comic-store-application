Pantalla para crear un cómic

Esta pestaña permite agregar, borrar o modificar un cómic.
Para que el cómic sea válido tiene que tener las siguientes características:

  • El nombre tiene entre 1 y 255 caracteres
  • El número de publicación está entre 1 y 999 999 999
  • Tiene que tener una colección
  • El número de publicación tiene que ser único para la colección
  • La descripción no tiene más de 512 caracteres

Si se usa como edición, la opción de borrar estará disponible
y también visualizará los ejemplares y autores asociados

Para poder crear, editar y borrar los ejemplares y autores
que tiene asociados un cómic hay dos tablas una tabla para ejemplares
y otra para autores.

Al lado de cada una de las dos tablas se encuentran 3 botones,
Nuevo, Editar y Borrar, los cuales permiten crear, modificar y eliminar
ejemplares o autores asociados al cómic.
